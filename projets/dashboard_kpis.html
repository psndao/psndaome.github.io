Dashboard KPIs E-commerce

🎯 Problématique Business

La startup e-commerce n’a aucune visibilité sur ses performances. Excel ne suffit plus. Objectif : dashboard professionnel avec KPIs en temps réel.

📊 Dataset

  • Source : Online Retail Dataset
  • Enrichissement : Faker pour simuler données récentes
  • Volume : 541 909 transactions + simulation temps réel
  • Métriques : CA, panier moyen, conversion, cohortes

🏗️ Architecture Technique

Raw Data → ETL → Database → BI Tool → Dashboard
 ↓        ↓       ↓         ↓          ↓
CSV     Python   PostgreSQL  Power BI   DAX

📦 Livrables

  • Pipeline Python (pandas) pour nettoyer et préparer les données
  • Base PostgreSQL avec modèle en étoile
  • Dashboard Power BI (KPIs e-commerce)
  • Mesures DAX avancées
  • Version alternative sur Tableau Public

🎯 Critères de réussite

  • Chargement du dashboard en moins de 3 secondes
  • Actualisation automatique quotidienne
  • KPIs clairs : CA, conversion, panier moyen, récurrence

🛠️ Stack utilisée

  • ETL : Python, pandas, SQLAlchemy
  • Base de données : PostgreSQL ou MySQL
  • BI : Power BI Desktop (gratuit) ou Tableau Public
  • Langages : SQL, DAX, Python
