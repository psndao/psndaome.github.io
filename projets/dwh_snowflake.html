❄️ Snowflake Data Warehouse & Analytics

Objectif du Projet

Construire une solution analytique moderne et scalable, de l’ingestion de données brutes jusqu’à la génération d’insights métiers via un entrepôt de données Snowflake (ou SQL Server), en suivant les meilleures pratiques : architecture Medallion, modélisation en étoile, documentation technique.

Structure du Répertoire

  • datasets/ : Données brutes (ERP, CRM, S3)
  • docs/ : Architecture, ETL, data catalog, nomenclature
  • scripts/ : SQL structurés en couches (bronze, silver, gold)
  • tests/ : Requêtes de validation qualité

Architecture de Données : Medallion

  • Bronze : données brutes (CSV) sans transformation
  • Silver : nettoyage, standardisation, mapping
  • Gold : tables de faits/dimensions prêtes pour l’analyse

Analyse & Visualisation

Le modèle final est interrogé via Power BI / Tableau pour générer des dashboards orientés métier :

  • Analyse des ventes par produit, canal, client
  • Segmentation, fréquence, récence
  • Tendance des performances dans le temps

Points Forts

  • Pipeline ETL structuré (bronze → gold)
  • Modélisation étoile claire et documentée
  • Documentation professionnelle (draw.io, markdown)
  • Compatible Snowflake et SQL Server

Stack Technique

  • Langages : SQL (T-SQL ou SnowSQL), YAML
  • Base de données : Snowflake ❄️ ou SQL Server
  • BI : Power BI, Tableau
  • Design : draw.io
