❄️ Snowflake Data Warehouse & Analytics

🎯 Objectif du Projet

Concevoir un entrepôt de données moderne et scalable avec Snowflake, selon l’architecture Medallion (Bronze → Silver → Gold), permettant une analyse métier puissante via Power BI ou Tableau.

📁 Structure du Répertoire

  • datasets/ : Données ERP, CRM, ou S3
  • docs/ : Architecture, catalogues, modèles
  • scripts/ : SQL pour chaque couche ETL
  • tests/ : Scripts de validation qualité

🧱 Architecture Medallion

  • 🥉 Bronze : Ingestion brute
  • 🥈 Silver : Nettoyage & transformation
  • 🥇 Gold : Modèle en étoile prêt pour le BI

📊 Analytique & Visualisation

Exploitation via Power BI / Tableau pour explorer :

  • Performance produit, région, canal
  • Analyse RFM et segmentation
  • Tendances temporelles des ventes

✅ Compétences mises en œuvre

  • 📦 Intégration multi-sources (ERP, CRM, S3)
  • 🏗️ Modélisation étoile & Medallion
  • 🔄 Pipeline ETL structuré
  • 🧪 Tests d’intégrité et qualité
  • 📊 KPI & requêtes analytiques
  • 📝 Documentation complète

🛠️ Stack Technique

  • Langages : SQL (SnowSQL / T-SQL), YAML
  • Base : Snowflake ou SQL Server
  • BI : Power BI, Tableau
  • Design : draw.io
